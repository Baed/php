



			Php sorting of arrays


	The elements in an array can be sorted in alphabetical or numerical 
	order desending and ascending.



		Ascending A-Z small to large
		Descending Z-A large to small


	Sort Functions For Arrays

	sort()- sort array in ascending order
	rsort()- sort array in decending order
	asort()- sort assocaitive array in ascending order according to value.
	ksort()- sort associative array in ascending order according to key.
	arsort()- sort associative array in desending order according to the value.
	krsort()- sort associative array in desending order according to the key.
